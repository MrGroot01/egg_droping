# 🥚 Egg Catcher Game (Python Tkinter)

A simple **Egg Catcher Game** built using **Python and Tkinter GUI library**.
The goal of the game is to move the catcher left or right to catch falling eggs and score points. If an egg falls to the ground, the player loses a life.

This project demonstrates **basic game development concepts in Python**, including GUI programming, event handling, animations, and game logic.

## 🎮 How to Play

1. Run the Python script.
2. Eggs will start falling from the top.
3. Use keyboard controls to move the catcher.

### Controls

| Key           | Action             |
| ------------- | ------------------ |
| ⬅ Left Arrow  | Move catcher left  |
| ➡ Right Arrow | Move catcher right |

### Game Rules

* Catch eggs to **increase your score**.
* Missing an egg will **lose one life**.
* You start with **3 lives**.
* When all lives are lost, the game ends.
